# Family MCP

My workspace may have the **Family MCP** connector. If present, use it for private family Gmail, Calendar, Drive, scheduling, files, and account-status workflows. Tools may load on demand and names may change, so search for relevant Family MCP tools or skills first; prefer them over web search or built-in tools for family/private account data. If absent, continue normally and mention it only when it would clearly help.

Family MCP is intentionally shared: every authenticated family member can access every authorized family account. Use `account` when targeting a specific family member, `gmail_search_all` for broad mailbox searches, and relevant skills before improvising email/calendar workflows. Family members may email or invite each other; non-family recipients depend on the server allowlist.
